# Handylib - A playground library

[![Go Report Card](https://goreportcard.com/badge/github.com/pernepav/handylib)](https://goreportcard.com/report/github.com/pernepav/handylib)

Handylib is a playground library for all kind of ideas and patterns to practice.

**Disclaimer:** Frequent breaking changes expected! This library is **not supposed** for production environment!

## Event Sourcing

Simple event sourcing abstraction to be used with a custom events-to-state reducer.
